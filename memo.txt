-------- 리녹스 명령어 기초연습 ---------
터미널 ctrl + j - > git Bash
1. ~ 현재 가장 최상위 디렉터리 위치 
2. pwd 작업폴더 현재 경로 디렉터리 표시
3. ls 현재 디렉터리 폴더와 파일 표시
4. mkdir 생성디렉터리명
    현재경로 위치의 디렉터리 생성
5. rm -r 삭제디렉터리명
    현재경로 위치에 디렉터리 제거(내용까지 모두)
6. cd 디렉터리명
    하위 해당 디렉터리로 이동
7. cd .. 
    상위 디렉터리는 이름을 인식할 수 없으므로
    한단계 위(..)라는 뜻으로 ..을 사용해서 나간다.
8. vim 생성파일명.확장자
    원하는 파일을 확장자 포함해서 ex)test.txt 생성 후
    i 눌러서 끼워넣기 버전으로 변경하고
    원하는 내용 입력 후 esc 키 누르고 :wq 저장하고 종료
9. cat 확인파일명.확장자
    내용 확인하고 싶은 파일명 확장자 포함 es)test.txt 작성 후
    확인하면 내용을 미리볼 수 있다.

Ctrl + C 또는 q를 누르면 정상으로 돌아온다
